# NetBox Device and Infrastructure Templates

This repository contains a collection of **NetBox device types, sites, and other infrastructure templates**.  
It is intended as a reference for network automation, documentation, and infrastructure management.

---

## Overview

- **Devices**: YAML definitions of network devices.
- **Device Types**: Templates including models, interface templates, and other device specifications.
- **Sites**: Representing data centers, offices, or other locations.
- **Platforms / Roles / Manufacturers**: Supporting objects for proper NetBox imports.
- **Images**: Illustrative images of device types for documentation purposes.
